ManGo, the Manchester Go user group

Wednesday 12th October 2016 - Building APIs in Golang

This month Edo Scalafiotti will be talking on how to build a serverless REST API using Go on AWS Lambda.

We will also have talks by Dan Bond and Tim Blackwell on subjects to be decided.

Sponsorship this month is from Sainsburys who are currently hiring (among other things) Go programmers in Manchester.
