The talk

The talk this month is by Lewis Marshall.

Containerised deployments with Flynn

Flynn is an open source platform aiming to simplify the deployment of distributed applications and services, primarily written in Go. I plan to talk about some of the many components which make up Flynn, and do a quick demo of actually running and deploying to a Flynn cluster.

About the Speaker

Lewis Marshall is a freelance coder who mainly writes Ruby web apps, but has recently got into Go through contributing to Flynn and is now a full time maintainer of the project.
